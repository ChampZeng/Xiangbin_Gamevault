# 🎮 Xiangbin's Game Vault: 系统拆解与策划改良作品集

欢迎来到我的个人游戏设计资产库。

本仓库收录了我对当前市场上头部商业游戏的深度逆向工程与系统分析。作为一个拥有极高游戏投入度（如《和平精英》KD 10+、《金铲铲之战》常驻高分段、《英雄联盟》数千小时游玩）的资深玩家与分析师，我始终致力于透过“好玩”的表象，去拆解其底层的**数值骨架、经济循环、玩家心理与商业化逻辑**。

这里不仅有系统层面的诊断，更包含了针对游戏现存痛点的**可落地策划改良方案**。

## 🎯 核心研究维度 (Core Analytical Dimensions)

本作品集的所有分析均围绕以下四个核心专业维度展开：
1. **核心玩法与机制重构 (Core Gameplay & Mechanics)**：拆解战斗逻辑、TTK（击杀时长）、博弈机制与操作反馈。
2. **经济系统与数值链路 (Economy & Numerical Systems)**：剖析局内资源流转（如自走棋的经济转化引擎）与局外长线养成体系（如 MMO 的抗通胀闭环）。
3. **商业化与消费心理学 (Monetization & Player Psychology)**：深度比对大抽奖（Gacha）与直售（Direct Sales）模式背后的损失厌恶、沉没成本与社交货币化逻辑。
4. **痛点诊断与产品迭代 (Pain Points & Actionable Design)**：摒弃空谈，针对用户流失点提出具体的系统优化案（如匹配机制改良、UI交互降噪、破局玩法引入）。
